Creates a WebMap using Folium. This project was done as part of an Online Course: The Python Mega Course: Build 10 Real World Applications by Ardit Sulce Creates a WebMap using Folium and python. This project was done as part of an Online Course: The Python Mega Course: Build 10 Real World Applications by Ardit Sulce.

The program obtains data from a JSON file and uses the information to add point marker features to the map. It initially contains a Open Street Map basemap onto which the data is superimposed. Multiple markers are added to the map and popup windows pertaining to the features are added to the map and assigned colors based on population.

Command to run the code: python map.py
Make sure all the supporting files are in the same directory and open the HTML to view the result.
